Welcome to the HelloVideo Documentation. In this file you will find the requirements required to install the script, instructions on how to install the script, instructions on how to configure different parts of the script, and troubleshooting tips.

So, lets begin…

Requirements

The requirements for this script are pretty minimal, but each requirements must be installed for the script to be fully functional.

  • PHP >= 5.4
  • Mcrypt PHP Extension
  • OpenSSL PHP Extension
  • Mbstring PHP Extension

Most of these requirements are installed on most modern hosting plans. Please see our Premium Hosting link for our recommended solution for hosting.

Installation

Installing the script is very simple. Additionally be sure to checkout our quick start videos where you will be walked through step-by-step on the installation process. Follow these basic steps to get the HelloVideo script setup and installed on your server.

  • Upload all the files from the hellovideo.zip contents to the root folder of your website location. The files you upload to your root folder will look like the following: (note if you are installing the developer version there are a few more steps required, please see the quick start videos for more info)
  • After all the files have been uploaded to your server, you’ll need to create a new database for your script.
  • Next you’ll need to add those new database credentials by opening the .env file and adding your database host, name, username, and password, the file you will edit look like the following and you’ll be replacing localhost, database, username, and password accordingly:
DB_HOST=localhost
DB_DATABASE=database
DB_USERNAME=username
DB_PASSWORD=password
  • After adding your database credentials be sure to save the file and then visit yoursite.com and you should see the following install screen in your browser (if not all the requirements pass you may have to contact your hosting provider to make sure your server meets all the requirements):
  • Finally, Click ‘Visit Your New Site’ button and you should see your new HelloVideo site in front of you:
  • And Just as easy as that you’ve just setup your new Video site. (!!! IMPORTANT !!!, make sure to remove the install.php file from your server, this is a security risk if you do not remove this file)

Upgrading

Upgrading the script is very simple. To upgrade to the latest version of the script, simply replace the ‘application’ folder with the new ‘application’ folder and you’ll be upgraded to the latest version.

To upgrade in the developer version you’ll have to look at the changelog(http://hellovideoapp.com/changelog) and make the file changes from your version up to the current version.

Admin Login

After successfully installing the script on your site you will be able to login with the default admin credentials:

username: admin

password: password

After loggin in successfully you can hover over your username on the top right, and click on ‘My Profile’:

Then click edit and you’ll be on your edit profile page:

Simply change your username, email and password to your desired admin login credentials and click ‘Update Profile’ to update your settings.

Admin & Configurations

To visit the admin section of your site you can hover over your username on the top right menu and click on the ‘Admin’ button. In this section we will breifly go over the Admin functionality and configurations.

Admin Dashboard

Upon visiting your admin section of your site you will be on the admin dashboard:

On the dashboard you can get a quick overview of your sites total subscribers, new subscribers, total videos, total posts, and analytics.

To access the analtyics section you must connect with your Google Analytics account. If you do not have an account you can Go Here to signup for a Google Analytics account. After signing up you’ll need to add your website URL. You will receive a Google Analytics Code that we will use later in the Settings Section to add Google Analytics tracking to your video site.

Videos

The next section of your site admin is the Videos Page:

By default the demo content is added to your site. You can simply click the trash can icon to delete all of the videos from your video manager.

In the video section you will be able to add new videos, edit videos, delete videos, and search for videos.

Users

The third section is the users section. Here you will be able to view all the current subcribers of your site:

You can see the different types of accounts, such as ‘Admin’, ‘Demo’, or ‘Subscriber’ & the subscription status of each user. Additionally, you will be able to add, edit, or delete any user account.

Pages & Posts

HelloVideo also works as a Content Management System so you can easily add any pages or posts. The Pages and Posts section allows you to create, edit, and delete any pages or posts on your site:

By default the demo pages and posts are installed, but you can simply click on the delete button to remove all the demo pages and posts, or you may use them as placeholders to create your new pages and posts for your site.

In the Pages and Posts section you can also add/edit and delete Post Categories.

In the post categories section you can add new categories, arrange categories, edit categories or delete categories. To arrange the order of the categories you can simply drag and drop any of the categories above, below, or inside each other.

Main Menu

The next section is the Menu Items. The Menu Items section is the main menu to your website:

You can easily build the menu for your site by clicking ‘Add New’, or editing/deleting any existing menu item. You can also drag and drop all categories to organize the order of your menu.

Themes

In the themes section you can select different themes for your video site. The default theme will be the only theme available when you first install the script:

You can choose to download the multiple other themes in the downloads area of the HelloVideo portal. After downloading each theme you will want to unzip the theme and place the theme inside of the /content/themes folder for example, if you unzip the tubed theme it will be in a new folder called /tubed/ you will want to move that folder so that way it sits in /content/themes/tubed/

After adding new themes you can reload your themes section and you should see the new themes available:

To activate another theme simply click the ‘Activate’ button and that theme will become your sites active theme.

Settings

There are 3 different settings for your video site. They are the Site Settings, the Payment Settings, and the Theme Settings

Site Settings

In the site settings you can change things such as the name, description, logo, and much more:

The following is a list of the things you can change in your site settings

  • Name (This is the name of your site)
  • Description (The description will show in the meta description and a few other places on your site)
  • Logo (This will be the main logo for your site)
  • Favicon (this is the small favicon on the browser tab)
  • Demo mode (whether the site is in demo mode or not), - Enable https:// (this will need to be enabled when you are ready to accept payments from your users, you will need to get an SSL for you domain before enabling this setting ’see quick start videos to learn more)
  • System email (this will be the email that will show for system emails such as password reset)
  • Social Networks (All of your social networks)
  • Google Analtyics Tracking ID (This is the tracking number you will get when you sign up for a Google Analtyics account. This ID usually starts with UA-XXX. Adding this will enable Analytic tracking for the visitors of your site)
  • Google Analytics API Integration (This will be the API id for your Dashboard Analtyics, see quick start videos to learn more about this integration)
Payment Settings

This is where you will add your Stripe payment settings so that way you can accept payments for your subscribers:

You can set your payment settings to test mode or live mode when you are ready to take real payments. The Stripe payment system is used to integrate payment settings directly in your site. Setting this up is super easy to learn more about setting this up see the quick start videos.

Theme Settings

The theme settings contain many different settings that can be changed specifically for the theme. Each theme has a different set of settings that can be changed such as headline, colors, etc:
